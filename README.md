# Sistema de Cadastro de Pessoas

> **Aviso:** Todos os dados utilizados neste projeto (nomes, CPFs, endereços e profissões) foram gerados de forma fictícia e aleatória exclusivamente para fins de estudo, prática e demonstração de conceitos de banco de dados.

Projeto desenvolvido utilizando MySQL para praticar conceitos de banco de dados relacionais.

## Tecnologias

* MySQL
* SQL
* MySQL Workbench

## Funcionalidades

* Criação de banco de dados
* Criação de tabela
* Inserção de registros
* Consulta de dados
* Exclusão de registros
* Aplicação de restrições de integridade

## Conceitos Aplicados

* PRIMARY KEY
* AUTO_INCREMENT
* NOT NULL
* UNIQUE
* INSERT
* SELECT
* DELETE

## Como executar

1. Execute `create_database.sql`
2. Execute `insert_data.sql`
3. Execute `consultas.sql`
4. Execute `delete_data.sql`
